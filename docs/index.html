MiniSAM: Simple AI for Object Segmentation

Point at an object → MiniSAM finds its boundaries. Draw a box around an object → MiniSAM finds it. This is a small, easy-to-understand version of SAM (Segment Anything Model) built from scratch.

What Does MiniSAM Do?

MiniSAM is like a smart scissors tool in image editing software.

  • You click on an object or draw a box around it
  • MiniSAM understands what object you want
  • It automatically finds the exact boundaries of that object
  • You get a clean mask (silhouette) of just that object

How Does It Work? (Simple Version)

Step 1: Image Understanding

The AI looks at your image and creates a visual "understanding" of it. It notices shapes, colors, textures, and edges.

Step 2: You Give Instructions

You click on the object you want, or draw a box around it. This is your "instruction" to the AI.

Step 3: Find the Object

The AI uses your instruction to find exactly which pixels belong to that object. It traces the boundaries carefully.

Step 4: Return the Mask

You get a clean black-and-white image showing only the object you asked for. The object is white, everything else is black.

Architecture: The 4 Key Parts

MiniSAM is made of 4 simple building blocks working together:

1️⃣ Image Encoder

Takes your image and converts it into a special format the AI understands. Like scanning and understanding a photograph.

Input: Image → Output: Visual Understanding

2️⃣ Prompt Encoder

Converts your instruction (click or box) into something the AI understands. Like translating your gesture into AI language.

Input: Click/Box → Output: Instruction

3️⃣ Mask Decoder

The "smart brain" that combines what it sees (image) with what you want (instruction). It figures out which pixels belong to your object.

Input: Image + Instruction → Output: Boundaries

4️⃣ IoU Predictor

Checks the quality of the result. It predicts "how good" the mask is so it can pick the best one if multiple options exist.

Input: Mask → Output: Quality Score
