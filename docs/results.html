MiniSAM Results

This page explains the results in a way that a new reader can understand. MiniSAM was trained from scratch on the Oxford-IIIT Pet dataset and evaluated using common segmentation metrics.

What do these metrics mean?

IoU

IoU measures overlap. If the predicted mask and the real mask overlap perfectly, IoU is 1. If they do not overlap, IoU is 0.

Dice

Dice is another overlap score. It is often used in medical imaging and segmentation tasks.

Precision

Precision asks: when the model predicts object pixels, how many are actually object pixels?

Recall

Recall asks: out of all real object pixels, how many did the model successfully find?

Training setup

Dataset

  • Oxford-IIIT Pet dataset.
  • Pet images with segmentation masks.
  • Object class: pet foreground versus background.

Model setup

  • MiniSAM trained from scratch.
  • Prompts: box plus positive point.
  • Output: multiple mask candidates and an IoU score.

Training curves

Training curves show whether the model improved over time. Loss should go down. IoU should go up.

Loss curve: lower values mean fewer mistakes.

IoU curve: higher values mean better mask overlap.

A good training run usually shows validation IoU increasing and then becoming stable. Once validation IoU stops improving, more training may not help much.

When MiniSAM works well

  • The object is clearly visible.
  • The prompt is inside or around the correct object.
  • The object is not extremely tiny.
  • The background is not too confusing.
  • The image is similar to the training data.
Possible failure cases: very small objects, unclear boundaries, heavy occlusion, cluttered backgrounds, or images very different from the pet dataset.

What is special about this project?

Built from scratch

The model is trained directly for this project instead of relying on the full original SAM weights.

Prompt-based

The model learns image plus instruction to mask, not just image to mask.

Easy to study

Each component is small enough to understand and test separately.

Good results

The final segmentation metrics show strong overlap with the real masks.
