MiniSAM Results

Here are the results from training MiniSAM on the Oxford-IIIT Pet dataset. The model was trained from scratch for 50 epochs and achieved strong performance!

Final Performance

These are the final metrics after 50 epochs of training:

Metric Value What It Means
IoU (Intersection over Union) 0.8949 89.49% overlap between predicted mask and real mask. Excellent! ✅
Dice Score 0.9440 94.40% similarity in shape. Very good! ✅
Precision 0.9447 When model says "object", it's right 94.47% of the time. Very accurate! ✅
Recall 0.9442 Model finds 94.42% of all actual object pixels. Complete! ✅
Pixel Accuracy 0.9028 90.28% of all pixels classified correctly. Great! ✅

Summary: MiniSAM achieves 89% accuracy on pet segmentation. It's very accurate at finding object boundaries and classifying pixels correctly!

Training Information

Dataset Used

  • Name: Oxford-IIIT Pet Segmentation Dataset
  • Photos: 3,680 images of pet animals
  • Pets: Cats and dogs
  • For each image: RGB photo + binary mask (object vs background)

Training Setup

  • Epochs: 50 (went through all data 50 times)
  • Batch size: 32 images at a time
  • Learning rate: 0.0001 (how fast model learns)
  • Prompts: Box + positive point (generated from ground truth masks)
  • Output masks: 3 candidates per image (model picks the best)

Training Time

  • Total: ~2-3 hours on GPU
  • Per epoch: ~2-3 minutes
  • Hardware: NVIDIA GPU with 8GB+ memory

When Does It Work Well?

MiniSAM works especially well when:

  • ✅ The object is clearly separated from background
  • ✅ The object fills a decent portion of the image
  • ✅ Lighting is reasonable (not too dark or blown out)
  • ✅ Your prompt (click or box) is close to the object
  • ✅ You're working with similar objects as the training data (pets)

When it might struggle: Tiny objects, very cluttered images, objects touching the edge, or completely different objects from training data.

What's Special About These Results?

From Scratch

No ImageNet weights, no SAM pre-training. Everything learned directly from 3,680 pet photos.

Small Model

Small image encoder + simple decoder. Fast to train and easy to understand.

Prompt-Based

Unlike normal segmentation models, this learns Image + Instruction → Mask.

Educational

Every component tested separately with easy-to-read code.
